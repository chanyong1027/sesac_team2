# ACME Employee Handbook (v1)

Document: ACME Employee Handbook v1  
Policy Set: Security + HR  
Last Updated: 2025-06-01  

## 1. Password Policy (Policy ID: SEC-PASS-001)

Minimum length: 12 characters.  
Rotation: every 90 days.  
MFA: required for all admin dashboards.  

Notes:
This policy applies to ACME internal systems (SSO, admin portal, VPN).

## 2. Remote Work / VPN

When working outside the office, employees must use the ACME VPN.
VPN is required for access to internal Git and the staging database.

## 3. Incident Reporting

If you suspect an incident (data leak, credential compromise, malware):
1. Notify #security within 15 minutes.
2. Open an incident ticket with the subject "INCIDENT: <short summary>".
3. Do not delete logs or reboot servers without approval.
