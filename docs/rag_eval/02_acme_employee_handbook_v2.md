## 1. Password Policy (Policy ID: SEC-PASS-001)

Minimum length: 16 characters.  
Rotation: every 180 days.  
MFA: required for all admin dashboards and production access.  

Exceptions:
- Service accounts may use long-lived credentials if stored in the secret manager.
- Emergency break-glass accounts must be reviewed monthly.

## 2. Remote Work / VPN

VPN is still required, but production access additionally requires device posture checks.

## 3. Incident Reporting

Same as v1, plus:
- Attach the first 10 minutes of relevant logs to the incident ticket.
